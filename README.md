# GreenTrade

GreenTrade is a customer-to-customer e-commerce platform for buying and selling eco-friendly, recycled, pre-loved and new items.

## Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- JavaScript
- Bootstrap
- InfinityFree Hosting
- phpMyAdmin

## Main Features

- User registration and login
- Browse items
- View item details
- Add new item listings
- Manage my items
- Checkout
- View orders
- Admin dashboard
- User management
- Listing management
- Reports
- Audit logs
